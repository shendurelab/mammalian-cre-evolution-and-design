# Phylogenetic silhouette images

All PNG silhouettes in this directory were sourced from **[PhyloPic](https://www.phylopic.org/)**, a free repository of open-licensed silhouette images of organisms. Individual images on PhyloPic are released under Creative Commons licenses (CC0 / CC-BY / CC-BY-SA) — please consult the original entry on <https://www.phylopic.org/> for the specific license and attribution of each silhouette before redistributing or re-using in publications.

Used in figure scripts to annotate phylogenetic trees (e.g., `scripts/make_figure1.R`, `scripts/make_ext_data_fig2be.R`).
